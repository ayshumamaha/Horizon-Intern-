# Student Management System (File-Based Storage)

## Description
This is a Python-based Student Management System designed to store and manage student records efficiently using file-based storage (JSON). It performs CRUD operations and helps organize student information.

## Features
- Add Student
- View Student Records
- Search Student
- Update Student
- Delete Student
- File-Based Storage
- Duplicate ID Prevention
- Input Validation
- Data Persistence

## Technologies Used
- Python 3.x
- JSON File Handling
- Dictionaries
- Lists
- Functions
- Loops
- Conditional Statements

## How to Run
1. Install Python.
2. Open terminal / command prompt.
3. Run:
```bash
python Student_Management_System.py
