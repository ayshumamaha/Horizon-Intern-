# Command-Line Calculator with Advanced Math Operations

## Description
This is a Python-based Command-Line Calculator developed to perform both basic and advanced mathematical operations. The project helps users solve calculations efficiently through a simple command-line interface.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Modulus
- Power
- Square Root
- Percentage Calculation
- Input Validation
- Continuous Calculations
- Exit Option

## Technologies Used
- Python 3.x
- Math Module
- Loops
- Functions
- Conditional Statements
- Exception Handling

## How to Run
1. Install Python on your system.
2. Open terminal / command prompt.
3. Run:
```bash
python Command_Line_Calculator.py
